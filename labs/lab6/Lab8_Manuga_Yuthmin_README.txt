# Lab 8 - Extension of Lab 6 and 7 - CSCI 3172

An extension of the front end of the portfolio of lab 6 and 7, this works on integrating back end functionality using serverless functions. Features include fetching dynamic project and weather data, and a contact form system that saves and displays messages.

## Built With

* [React](https://reactjs.org/) - Frontend library
* [Vite](https://vitejs.dev/) - Frontend tooling
* [Node.js](https://nodejs.org/en/) - JavaScript runtime (for Netlify Functions)
* [Netlify Functions](https://docs.netlify.com/functions/overview/) - Serverless backend execution
* [Axios](https://axios-http.com/docs/intro) - Promise-based HTTP client (for frontend API calls and backend weather fetch)
* [React Router DOM](https://reactrouter.com/web/guides/quick-start) - Frontend routing
* [OpenWeatherMap API](https://openweathermap.org/api) - Source for weather data
* CSS - Styling (including CSS variables for theming)
